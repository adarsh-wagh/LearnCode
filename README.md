# LearnCode

LearnCode is a full-stack interactive coding platform designed to make programming education more engaging through hands-on practice. Unlike traditional learning platforms that rely primarily on video lectures and reading materials, LearnCode combines structured learning content with a real-time coding environment, allowing users to write, run, and test code directly in the browser.

The platform is built using Next.js, React, Tailwind CSS, Clerk Authentication, Neon PostgreSQL, and Drizzle ORM. Sandpack is integrated to provide an in-browser coding environment with instant code execution and feedback, helping users strengthen their programming and problem-solving skills.

https://learn-code-kohl.vercel.app/

## Features

* Interactive in-browser code editor
* Real-time code execution with Sandpack
* Secure user authentication with Clerk
* Structured coding lessons and exercises
* Progress tracking and achievement monitoring
* Responsive and modern user interface
* PostgreSQL database powered by Neon
* Type-safe database operations using Drizzle ORM

## Tech Stack

* Next.js
* React
* Tailwind CSS
* Clerk
* Neon PostgreSQL
* Drizzle ORM
* Sandpack

## Project Goal

The goal of LearnCode is to provide an interactive learning experience that bridges the gap between theory and practice by enabling learners to apply programming concepts directly within the platform.
